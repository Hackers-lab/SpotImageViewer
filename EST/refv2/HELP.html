📖 ERP Estimate Generator — User Guide

Version 5.0 — Interactive electrical network estimation tool for WBSEDCL projects.

1. Getting Started

1.1 First Launch

When you open the application for the first time, a Project Setup Wizard will appear. Fill in the project name, select the project type, and configure UH material preferences.

1.2 Project Types

1.3 UH Materials (NSC Only)

When "Use UH Materials" is checked in Project Settings, the estimate uses readymade UH brackets, pole clamps and D-iron clamps instead of raw steel (Angle + Flat) for LT pole brackets. This option is only available for NSC projects.

UH ModeItems Used
UH ON (Readymade)UH-LT BKT 4 WAY, UH-CLAMP (4/pole for ACSR, 2/pole for AB Cable), UH-Diron (per wire count)
UH OFF (Raw Steel)1m Angle 65×65×6mm + 1m Flat 65×6mm per LT pole

2. Drawing Tools

2.1 Canvas Objects

2.2 Creating a Span

  1. Select the SPAN tool from the toolbar.
  2. Click the first pole (it will highlight in green).
  3. Click the second pole to draw the span between them.
  4. The span length is auto-calculated from the pole positions. Edit in properties.
Tip: Span conductor type is auto-detected from connected poles: HT poles → ACSR, LT poles → depends on your selection (ACSR, AB Cable, or PVC Cable).

2.3 Detail View Toggle

Click "👁 Detail: ON/OFF" to show or hide stay wires, earthing symbols, cattle guard markers, and extension indicators on the canvas.

4. Live Estimate

The estimate table on the right updates automatically as you draw. It shows all materials and labour computed by the rule engine.

4.1 Editing Quantities

Double-click the Qty column of any row to manually override a quantity. Overridden values are preserved until you change the drawing.

4.2 Adding Custom Items

Use "+ Add Material" or "+ Add Labor" buttons to search and add items from the master database that aren't covered by rules.

4.3 3% Sag Allowance

The following items automatically get a 3% sag/wastage multiplier applied:

  • All steel sections (Channel, Angle, Flat)
  • GI Wire (4mm and 5mm)
  • ACSR Conductor (all sizes)
  • PVC Cable (10, 16, 25 SQMM)
  • AB Cable

5. Export

5.1 Excel Export

Click "📊 Generate Excel" or use File → Generate Excel. The workbook contains:

  • Estimate sheet — Full BOM with materials, labour, supervision, and grand total
  • Iron Breakup sheet — Detailed per-source breakdown of all steel and GI wire items

5.2 PDF Export

Click "🗺️ Export PDF" or use File → Export PDF to save the canvas drawing as a PDF file with legend.

6. Settings (Menu)

6.1 Master Database (Excel Sync)

Import or export the material and labour rate database from/to Excel. Useful for bulk rate updates.

6.2 Ruleset Manager

View, edit, add, or delete rules in the JSON ruleset that drives the estimate calculations. Each rule has:

  • Object — Which canvas object type it applies to
  • Condition — Python expression evaluated against the object context
  • Type — Material or Labor
  • Item Code / Name — Database item reference
  • Formula — Quantity calculation expression
Warning: Editing rules incorrectly can break the estimate. Always back up rules.json before making changes.

8. File Format

Projects are saved as .json files containing all pole, structure, span, and consumer data along with project metadata. The application auto-saves to autosave_erp.json on every change.

Tip: The app is fully backward-compatible with v4 saved files. Old files will be upgraded automatically when opened.

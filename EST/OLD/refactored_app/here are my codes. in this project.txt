Project: ERP Estimate Generator — a PyQt6 desktop application for electrical network project estimation used in WBSEDCL (West Bengal electricity utility) projects. The app has an interactive canvas for drawing network diagrams and generates live Bills of Materials (BOM) and labor cost estimates, exportable to PDF drawings and ERP-compatible Excel files.
Architecture: Multi-module Python app — app.py (main window), canvas_objects.py (SmartPole, SmartSpan, SmartHome as QGraphicsPathItem subclasses), rule_engine.py (JSON-driven dynamic BOM calculator using safe eval), database.py (SQLite with materials and labor tables), ui_components.py (InteractiveView, DraggableLabel), ui_dialogs.py (SearchDialog, SettingsDialog, DatabaseManagerDialog, RulesetManagerDialog), constants.py (TOOLS, PROPERTY_DATA, FORMULA_VARS), rules.json (all BOM rules).
Planned Redesign — implement in phases, do not break existing functionality:
Phase 1 — Project Setup Wizard: On app launch, show a setup dialog capturing: Project Name, Lat/Long, Project Type (NSC / FDS+TURNKEY / MAINTENANCE / SHIFTING / AUGMENTATION), UH Material Toggle (yes/no, project-level, not per-estimate), Division/Circle info. Project type determines supervision rate: 10% for NSC, 15% for all others. GST is fixed at 18% on labour only. UH toggle flows into rule engine context as use_uh global variable (already exists, just move to project level). Store all project metadata in a project_meta dict on the main app object. Make it editable via a "Project Settings" button after initial setup.
Phase 2 — SmartPole property updates: Keep LT/HT distinction from canvas tool selection. Add pole_type2 property: PCC (default), STP (Steel Tubular Pole), H-BEAM. Add cascading height selection — PCC: 8MTR/9MTR, STP: 9MTR/9.5MTR/11MTR, H-BEAM: 13MTR. Default height: 8MTR for LT, 9MTR for HT. Add extension_required boolean and extension_height numeric (default 3MTR, only visible when extension checked). Keep earth count (default 1) and stay count (auto from vector angle calculation with manual override — show "🔒 Manual" indicator when overridden, add "Reset to Auto" button). The auto-stay logic: 1 span connected = 1 stay, 2 spans with turning angle >20° = 1 stay, all else = 0.
Phase 3 — SmartStructure as new separate canvas object: Separate from SmartPole entirely. Always HT-side. Properties: Structure Type (DP/DTR/TP/4P), Pole Type (PCC/STP/H-BEAM, default PCC), Height (cascading per pole type, default 9MTR), Extension (yes/no + height in metres), Earth Count (editable, defaults: DP=2, TP=3, 4P=4, DTR=5), Stay Count (editable, default 4 for all structure types), DTR Size dropdown (10KVA/16KVA/25KVA/63KVA/100KVA/160KVA — only visible when Structure Type = DTR). Canvas symbol: DP=2 circles, TP=3 circles, 4P=4 circles, DTR=2 circles with transformer symbol. Earth and stay symbols drawn attached to the structure symbol on canvas.
Phase 4 — SmartSpan property updates: Voltage level auto-detected from connected pole types (LT pole connected = LT span, HT pole = HT span). Conductor type: ACSR (HT default), ABC (LT default), UG (underground, future), PVC. Conductor sizes — ACSR: 30SQMM/50SQMM. ABC LT: 3CX50+1CX35, 3CX50+1CX16+1CX35, 3CX70+1CX16+1CX50. ABC HT: 3CX50+1CX150, 3CX95+1CX70. PVC: 10/16/25/50/95/120 SQMM. Wire count for ACSR: 2/3/4. CG (Cattle Guard) remains a span property (yes/no). Length default 40m. CG symbol drawn as small crosshatch below span midpoint on canvas.
Phase 5 — SmartConsumer (previously SmartHome): Simplified. Properties: Phase (1PH/3PH), Cable Size dropdown (context-aware per phase), Agency Supply checkbox (WBSEDCL-supplied vs Agency-supplied material — reflected in Excel output as a note).
Phase 6 — Canvas symbol improvements: Stay wire = diagonal line at 45° from pole base with anchor point. Earth = 3 decreasing horizontal lines below pole (standard ⏚ symbol). CG = small crosshatch bracket below span midpoint. TP = 3 circles in triangle arrangement, 4P = 4 circles in square arrangement. Add "Detail View" toggle in toolbar to show/hide ancillary symbols (stay, earth, CG) for clean navigation vs detailed PDF export view.
Phase 7 — Iron Breakup Excel Sheet: Add a second sheet "Iron Breakup" to the generated Excel. Structure matches the provided breakup format with sections: B (MS Channel 75×40mm), C (MS Angle 65×65×6mm), D (MS Angle 50×50×6mm), E (MS Flat 65×6mm).
